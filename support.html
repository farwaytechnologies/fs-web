Frequently Asked Questions

How do I install Farway Security on multiple devices?

To install Farway Security on multiple devices, you can use our central management console. Log in to your account, navigate to Device Management, and select "Add New Device." You can then generate installation links or download the installer package for distribution. Our Enterprise plans include deployment tools for large-scale installations.

What should I do if I forgot my account password?

If you've forgotten your account password, visit our login page and click on "Forgot Password." Enter your registered email address, and we'll send you a password reset link. For security reasons, the link expires after 24 hours. If you don't receive the email, check your spam folder or contact our support team for assistance.

How often are security updates released?

We release regular security updates on a bi-weekly schedule to ensure your protection against the latest threats. Critical security patches may be released more frequently as needed. By default, Farway Security is configured to automatically download and install these updates, but you can customize your update settings through the application preferences.

Can I transfer my license to a new device?

Yes, you can transfer your Farway Security license to a new device. First, deactivate the license on your old device by going to Settings > License > Deactivate. Then, install Farway Security on your new device and activate it using your license key or account credentials. Each license allows installation on a specific number of devices based on your subscription plan.

How do I schedule a system scan?

To schedule a system scan, open the Farway Security application and navigate to the "Scan" tab. Click on "Schedule Scan" and select your preferred frequency (daily, weekly, or monthly), time, and scan type (quick, full, or custom). You can also configure additional options such as low-resource usage mode or actions to take when threats are detected.

Common Troubleshooting

Installation Issues

  1. Check System Requirements

    Verify that your system meets the minimum requirements for Farway Security. This includes operating system version, processor, RAM, and available disk space.

  2. Disable Conflicting Security Software

    Temporarily disable any existing security software that might interfere with the installation process. Multiple security products can sometimes conflict with each other.

  3. Run as Administrator

    Right-click the installer and select "Run as administrator" to ensure the installation has the necessary permissions.

  4. Use Clean Boot

    If you're still experiencing issues, try performing a clean boot to eliminate potential software conflicts during installation.

Performance Optimization

  1. Adjust Scan Settings

    Configure scan settings to run during periods of low system usage. You can schedule scans during off-hours or enable the "Low Priority" option.

  2. Exclude Trusted Files

    Add frequently accessed files and folders to the exclusion list to prevent unnecessary scanning of trusted content.

  3. Update Software

    Ensure you're running the latest version of Farway Security, as performance improvements are regularly included in updates.

  4. Optimize System Resources

    Close unnecessary applications and services to free up system resources for Farway Security to operate efficiently.
